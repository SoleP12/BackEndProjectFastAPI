# BackEndProjectFastAPI

cd ./backend/
python -m venv .venv
source ./venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

This is a personal project built with FastAPI that demonstrates how to build a clean and scalable REST API. The API allows users to create, update, and delete tasks with user authentication. The project was created to practice FastAPI, JWT authentication, and database integration.
